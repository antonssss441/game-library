# Игровая библиотека

Проект представляет собой Python-script, который использует APi базы данныз видеоигр RAWG для  отображения названия игр, дату выхода, ссылки на игры, скриншоты из игр и магазины, где их можно купить.

## Как установить

1. Убедитесь, что у вас установлен Python 3.13.3 или выше.
2. Установите зависимости. Чтобы воспользоваться библиотекой пропишите `pip install -r requirements.txt`
3. Скачайте или склонируйте репозиторий с этим скриптом.
4. В командной строке перейдите в папку с скриптом.

## Как запустить 

для запуска программы введите в командной строке `python main.py`.
